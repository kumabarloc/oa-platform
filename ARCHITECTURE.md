# OA 平台 - 技术架构文档

> 最后更新：2026-03-30

## 系统架构
- **前端：** Vue 3 + Element Plus（SPA）
- **后端：** Spring Boot 3.x + JDK 21
- **数据库：** MySQL 8.x
- **部署：** 待定

## 前端架构
- Vue 3（Composition API）
- Element Plus UI 组件库
- Vue Router（路由）
- Pinia / Vuex（状态管理）
- Axios（HTTP 请求）

## 后端架构
- Spring Boot 3.x
- Spring Security + JWT（认证鉴权）
- MyBatis / MyBatis-Plus（持久层）
- Flyway（数据库迁移）
- Maven（多模块项目）
